# Relativity-style Export

This folder contains a simple exporter that writes:

- `loadfile.dat` (Concordance DAT)
- `images.opt` (stub, points to TXT as "images" for demo)
- alongside `out/relativity/natives/` and `out/relativity/text/` if you copy them here.

## Usage

1. Run the pipeline first:
   ```bash
   python run_pipeline.py --input 00-setup/sample-data --output out
   ```
