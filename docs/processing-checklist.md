# Processing Checklist (Quick)

- [ ] Inventory created; sources logged
- [ ] Hashing algorithm selected (MD5 and/or SHA256) and captured
- [ ] deNIST completed; exceptions documented
- [ ] Deduplication: ☐ Global ☐ Custodian ☐ Family (document choice)
- [ ] Timezone normalization set and recorded
- [ ] Metadata extracted; spot-check (10–20 docs) vs natives
- [ ] Email threading keys extracted (MessageID/ThreadID as available)
- [ ] Exceptions/containers (PST/ZIP/RAR) tracked and resolved or flagged
- [ ] Text extraction complete; OCR policy applied
- [ ] Sample load file validated (field counts, delimiters)
