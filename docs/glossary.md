# Glossary (Quick)

- **deNIST** – removal of known system files per NIST list.
- **Deduplication** – removal of duplicate documents by hash; scopes: global/custodian/family.
- **DAT/OPT** – metadata and image cross-reference load files for review platforms.
- **Inclusive Email** – the most complete email in a thread (contains prior content).
- **Placeholder** – a stub image page indicating a native was produced instead of an image.
