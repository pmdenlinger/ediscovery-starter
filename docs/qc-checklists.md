# QC Checklists (Processing / Review / Production)

## Processing QC
- [ ] Hash inventory matches collection log counts
- [ ] Deduplication documented; sample verified
- [ ] Metadata spot-check: 10–20 items vs native
- [ ] Exception report reviewed and cleared/flagged
- [ ] Test ingest of DAT/OPT/Text/Natives succeeded

## Review QC
- [ ] Sampling for coding consistency (stratified by custodian/date/issue)
- [ ] Privilege sample spot-check (counsel comms, in-house vs outside)
- [ ] Redaction placement and reason codes verified

## Production QC
- [ ] Bates sequences continuous; Beg/End families intact
- [ ] Image and text path checks (random sample)
- [ ] Natives only where specified; placeholders correct
- [ ] Load files validated in sandbox workspace
- [ ] Checksums created and recorded
